# @softcodefr/vue-lens-core

Shared core package for Vue Lens.

It exposes the in-memory event collector and debug event types used by [`@softcodefr/vue-lens`](https://www.npmjs.com/package/@softcodefr/vue-lens).

## Exports

```ts
import { collector } from '@softcodefr/vue-lens-core'
import type { DebugEvent, RenderEvent } from '@softcodefr/vue-lens-core'
```

## Scripts

```bash
pnpm build
pnpm typecheck
pnpm dev
```
